# 🎮 俄罗斯方块游戏

一个现代化的网页版俄罗斯方块游戏，具有流畅的操作体验和丰富的功能特性。

## ✨ 功能特点

- 🎯 三种难度模式（简单、中等、困难）
- 🎵 背景音乐控制
- 📱 支持移动端操作
- 🎨 精美的方块颜色
- 📊 实时分数统计
- ⚡ 动态难度提升

## 🎮 操作说明

### ⌨️ 键盘控制

| 按键 | 功能 |
|------|------|
| ↑ | 旋转方块 |
| ← | 向左移动 |
| → | 向右移动 |
| ↓ | 加速下落 |
| 空格 | 直接落地 |

### 📱 触屏控制

移动端用户可以使用屏幕上的虚拟按钮进行操作：

- 左箭头：向左移动
- 右箭头：向右移动
- 旋转：旋转方块
- 下箭头：加速下落

## 🎚️ 难度模式

### 简单模式
- 初始速度较慢
- 速度提升幅度小
- 得分倍率：1.0x

### 中等模式
- 中等初始速度
- 适中的速度提升
- 得分倍率：1.5x

### 困难模式
- 较快的初始速度
- 明显的速度提升
- 得分倍率：2.0x

## 🚀 如何开始

1. 打开游戏页面
2. 选择难度模式
3. 点击"开始游戏"按钮
4. 可选：点击音乐按钮开启背景音乐

## 🎯 游戏目标

- 控制下落的方块
- 完成整行消除得分
- 通过消除行数提升等级
- 在速度不断提升的挑战中获得高分

## 💡 游戏提示

- 注意观察"下一个方块"提示
- 合理使用"直接落地"功能可以提高得分速度
- 随着等级提升，方块下落速度会逐渐加快
- 合理规划方块摆放位置，为长条形方块预留空间

## 🎵 音乐控制

- 点击音乐按钮开启/关闭背景音乐
- 使用音量滑块调节音乐大小
- 游戏暂停时音乐会继续播放

## 🏆 计分规则

- 基础分数：每消除一行获得100分
- 等级加成：当前等级会作为分数倍率
- 难度加成：不同难度模式有不同的分数倍率
- 最终分数 = 基础分数 × 等级 × 难度倍率
